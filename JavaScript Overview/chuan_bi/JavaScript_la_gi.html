I. JavaScript

  • JavaScript là một ngôn ngữ lập trình được sử dụng để tạo trang Website tương tác.Từ làm mới bảng tin trên trang mạng xã hội đến hiển thị hình ảnh động và bản đồ tương tác, các chức năng của JavaScript có thể cải thiện trải nghiệm người dùng của trang web. Là ngôn ngữ kịch bản phía máy khách, JavaScript là một trong những công nghệ cốt lõi của World Wide Web.
  • JavaScript là một ngôn ngữ script được dùng để tạo các script ở máy client và máy server. Các script ở máy client có thể thực thi được trong trình duyệt trong khi các script ở máy server lại thực hiện trên server.
  • JavaScript là một ngôn ngữ script dựa trên đối tượng nhằm phát triển các ứng dụng dựa trên client và server

II. Ứng dụng của JavaScript

  • Lập trình web front end với các framework hàng đầu: JQuery, React, Angular, VueJS
  • JS còn được sử dụng dể tạo các ứng dụng ở phía back-end với NodeJS
  • Lập trình viên ứng dụng di động với React Native, Inoic, NativeScript
  • Lập trình game với Unity
  • Lập trình robot, IoT

V. Cú pháp Javascript

Cú pháp JS bao gồm những quy tắc viết mã JS. Chúng ta phải tuân thủ cú pháp khi xây dựng một chương trình. Một chương trình máy tính(computer program) là một danh sách các hướng dẫn (instruction) được máy tính thực thi(execute). Trong các ngôn ngữ lập trình các hướng dẫn này được gọi là câu lệnh. Và ngăn cách bằng dấu chấm phẩy.

  1. Literal
    2. là giá trị cố định không thay đổi được. VD 10; 10.5; "Hà Nội",...
  2. Biến (Variable)
    3. được sử dụng để lưu trữ giá trị
    Một biến cần được khai báo trước khi sử dụng var pass; khai báo một biến (sử dụng var hoặc let) pass = 10; gắn giá trị cho biến
  3. Toán tử
    4. chẳng hạn như toán tử số học, toán tử so sánh, toán tử logic
  4. Biểu thức
    5. là kết hợp của giá trị, biến, toán tử
  5. Từ khóa
    6. là các từ được quy định sẵn của ngôn ngữ lập trình
  6. Chú thích
    7. Sử dụng 2 dấu gạch chéo (một dòng) hoặc /* */ (cho nhiều dòng)
