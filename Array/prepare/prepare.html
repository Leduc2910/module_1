Mảng là gì?

  • Mảng là một loại biến đặc biệt, có thể lưu được nhiều giá trị thay vì chỉ một giá trị như các biến thông thường
  • Mỗi giá trị trong mảng được gọi là một phần tử
  • Các phần tử được lưu trữ ở các vị trí kế tiếp nhau trong bộ nhớ

Các khái niệm của mảng

  • Tên mảng: Tuân thử theo quy tắc đặt tên biến
  • Phần tử: các giá trị được lưu trữ trong mảng
  • Chỉ số: Vị trí của các phần tử trong mảng. Chỉ số bắt đầu từ 0
  • Độ dài: số lượng các phần tử của mảng

Khai báo mảng

  • Cách 1: 
        var array_name = [item1, item2,....];

    var cars = ['Saab', 'Volvo', 'BMW'];
  • Cách 2: Sử dụng từ khóa new
    •     var array_name = new Array(item1, item2,....);

    var cars = new Array('Saab', 'Volvo', 'BMW');

Phần tử, chỉ số và độ dài của mảng

  • Mỗi phần tử (item) được xác định bằng một số thứ tự còn gọi là chỉ số (index) duy nhất trong mảng
  • Chỉ số là một số nguyên dương
  • Chỉ số của phần tử đầu tiên là 0
  • Chỉ số của phần tử cuối cùng là n-1, trong đó n là độ dài của mảng
  • Có thể truy xuất đến phần tử của mảng thông qua chỉ số
  • Truy xuất phần tử của mảng
    • Mỗi phần tử trong mảng được thao tác giống như một biến
    • Truy xuất các phần tử thông qua chỉ số đặt trong dấu []
    • Ví dụ, giá giá trị cho phần tử đầu tiên của mảng: 
                    cars[0] = "Opel";
    • Ví dụ, lấy giá trị của một phần tử đầu tiên: 
                    var name = cars[0];
  • Thuộc tính length là độ dài của mảng 
        Ví dụ:
      var fruits = ['Banana', 'Orange', 'Apple', 'Mango'];
      alert(fruits.length);
      Thuộc tính:
          fruits: tên mảng
          Banana, Orange, Apple, Mango : là các phần tử trong mảng, lần lượt các chỉ số trong mảng là 0,1,2,3
          fruits.length : độ dài của mảng ở đây là 4 vì có 4 phần tử

Sử dụng for, for-in để duyệt mảng

  • Vòng lặp for-in (còn gọi là enhanced for) được sử dụng để duyệt qua các phần tử của một collection, chẳng hạn như mảng, danh sách,...
  • Cú pháp: 
            for (let element in array) {
            console.log("element = "+array[element]);
        }
  • Sử dụng vòng lặp for để duyệt qua tất cả các phần tử của mảng
  • Cú pháp: 
            for (let i=0;i < array.length;i++) {
            console.log("element="+array[i]);
  • Khác biệt khi sử dụng for và for-in:
    • Có thể sử dng index khi duyệt bằng for, không có index nếu sử dụng for-in
    • Có thể sử dụng for để duyệt theo các chiều khác nhau(từ đầu đến cuối, từ cuối đến đầu,..)
    • For in chỉ duệt từ đầu đến cuối mảng

Thêm phần tử vào mảng: push(), length

  • Phương thức push() cho phép thêm phần tử vào cuối mảng
  • Ví dụ: 
            var fruits = ['Banana', 'Orange', 'Apple', 'Mango'];
        fruits.push("Lemon");
  • Có thể thêm phần tử vào mảng bằng cách gán giá trị cho phần tử ở vị trí length
  • Ví dụ: 
            var fruits = ['Banana', 'Orange', 'Apple', 'Mango'];
        fruits[fruits.length] = "Lemon";

join()

  • Phương thức join() chuyển tất cả các phần tử trong mảng thành chuỗi và nối chúng lại với nhau
  • Ví dụ: 
            var a = [1, 2, 3];      // Tạo một mảng mới với 3 phần tử
        a.join()                // => "1,2,3"
        a.join(" ")             // => "1 2 3"
        a.join("")              // => "123"
        var b = new Array(10);  // Một mảng có chiều dài là 10 với không phần tử
        b.join('-')             // => '---------' một chuỗi với 9 dấu gạch ngang

reverse()

  • Phương thức reverse() đảo ngược các phần tử trong mảng, trả về một mảng các phần tử bị đảo ngược
  • Ví dụ: 
            let a = [1,2,3];
        a.reverse().join()      // => "3,2,1" và a = [3,2,1]

sort()

  • Sắp xếp các phần tử của mảng sử dụng phương thức sort()
  • Ví dụ: 
            let a = new Array("banana", "cherry", "apple");
        a.sort();
        let s = a.join(", ")        // s = "apple, banana, cherry"

concat()

  • Nối các phần tử của hai mảng với nhau. Trả về mảng mới chứa phần từ của mảng gốc và mảng được nối
  • Ví dụ: 
            let a = [1,2,3];
        a.concat(4, 5)                // return [1,2,3,4,5]
        a.concat([4,5]);              // return [1,2,3,4,5]
        a.concat([4,5],[6,7]);        // return [1,2,3,4,5,6,7]
        a.concat(4,[5,[6,7]])         // return [1,2,3,4,5,[6,7]]

push() và pop()

  • Hai phương thức push và pop hoạt động giống cấu trúc ngăn xếp (FIFO):
    • Phương thức push() để thêm một hoặc nhiều phần tử vào cuối mảng
    • Phương thức pop() xóa đi phần tử cuối cùng của mảng, làm giảm kích thức và trả về giá trị của phần tử bị xóa
  • Ví dụ: 
            let stack = [];             // stack[]
        stack.push(1,2);            // stack[1,2] return 2
        stack.pop()                 // stack[1] return 2
        stack.push(3)               // stack[1,3] return 2
        stack.pop()                 // stack[1] return 3
        stack.push([4,5])           // stack[1,[4,5]] return 2
        stack.pop()                 // stack[1] return [4,5]
        stack.pop()                 // stack[]  return 1

shift() và unshift()

  • Phương thức shift() để xóa phần tử vào đầu mảng
  • Phương thức unshift() để nối phần tử đầu tiên của mảng
  • Ví dụ: 
            let a = [];
        a.unshift(1)           //a[1]   return 1
        a.unshift(22)          //a[22,1]    return 2
        a.shift()              // a[1]       return 22

toString()

  • Phương thức chuyển toàn bộ các phần tử trong mảng sang dạng chuỗi
  • Ví dụ: 
            [1,2,3].toString()          // Yields '1,2,3'
        ['a', 'b', 'c'].toString()  // Yields 'a,b,c'
        [1,[2,'c']].toString()      // Yields '1,2,3'
